# School Website Project

## Description
An advanced web-based assistant platform for students, featuring ChatGPT-level conversational intelligence, multi-layer intent detection, adaptive responses, and session-based context awareness. The project helps users explore STEM programs, admissions, and writing resources through a modern, interactive interface.

---

## üöÄ Setup & Installation
1. Clone this repository.
2. Install Python dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3. Run the app:
    ```bash
    python app.py
    ```

---

## üí° Usage
Open your browser and go to `http://localhost:5000` (or the configured port). Interact with the assistant via the web interface.

---

## ‚ú® Features
- ChatGPT-level semantic intent detection
- Multi-layer intent detection (category & type)
- Context-aware inference for natural conversations
- Adaptive, session-based responses
- STEM program exploration and admissions guidance
- Writing resources and tips
- Flexible, typo-tolerant input handling
- Modern web interface

---

## Technical Highlights
- JavaScript (Vanilla, no frameworks)
- Integrates with Flask backend
- No external APIs, instant responses
- Accessibility: ARIA labels, prefers-reduced-motion
- Browser compatibility: All modern browsers
- Session memory: Current tab only

---

## Safety & Boundaries
All previous safety features preserved:
1. Admissions Safe Mode: No predictions or guarantees, redirects to official Ministry sources
2. Honest Uncertainty: Admits when it doesn't know, points to resources
3. Decision Boundaries: Provides options, not commands
